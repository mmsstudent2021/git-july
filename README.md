- git init | git စသုံးမယ်ကြေညာ
- git remote add origin git_hub_url | github က repo နဲ့ချတ်

- git add . | အပြောင်းအလဲအကုန်ထည့်
- git commit -m "message" | ဘာပြောင်းထားလဲ commit (message ရေး)
